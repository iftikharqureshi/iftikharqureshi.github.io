Portfolio Projects 

1- Image Generation with Stability AI

A Python notebook demonstrating text-to-image generation and image inpainting using the Stability AI API. The project shows how prompt engineering and masking can be used to iteratively refine generated images.

https://github.com/iftikharqureshi/image_generation_stability_ai

2- Retrieval-Augmented Generation (RAG) Mini Demo

A compact implementation of a retrieval-augmented question-answering pipeline that embeds local documents, retrieves relevant context, and generates grounded answers using a language model.

https://github.com/iftikharqureshi/rag_mini

3- Titanic Exploratory Data Analysis

An exploratory data analysis of the Titanic dataset focusing on data cleaning, missing value handling, visualization, and identifying factors associated with passenger survival.

https://github.com/iftikharqureshi/eda_titanic

4- Supervised Fine-Tuning for Structured JSON Output

A supervised fine-tuning project that trains a language model to convert dense paragraphs into structured JSON summaries, bullet points, and vocabulary terms using a custom JSONL dataset.

https://github.com/iftikharqureshi/supervised_fine_tuning

5- Plain-English Explainer Application

A Streamlit application that rewrites complex academic or technical text into structured, plain-English explanations using a fine-tuned language model and JSON schema validation.

https://github.com/iftikharqureshi/plain_english_explainer

6- Chatbot with MLflow Tracking

A console-based chatbot that logs latency, token usage, and model parameters to MLflow, demonstrating how experiment tracking can be applied to generative AI workflows.

https://github.com/iftikharqureshi/chatbot_mlflow

7- Image Classification with Convolutional Neural Networks (MNIST)

A deep learning project that trains a convolutional neural network on the MNIST handwritten digits dataset and evaluates performance using accuracy metrics and confusion matrices.

https://github.com/iftikharqureshi/image_classifier

8- Emoji-Style Chatbot Fine-Tuning

A fine-tuning experiment where a language model is trained to respond exclusively using emoji labels, illustrating dataset design, constrained outputs, and model behavior control.

https://github.com/iftikharqureshi/emoji_fine_tuning

9- Global Wellbeing Data Explorer

An exploratory analysis that combines multiple global datasets (health, economic, education indicators) using SQL and Python to examine relationships across countries and time.

https://github.com/iftikharqureshi/wellbeing_explorer

10- Invoice Parsing and Structured Data Extraction

A Python notebook that processes invoice PDFs and extracts structured JSON fields using a large language model, demonstrating document understanding and data extraction techniques.

https://github.com/iftikharqureshi/parse_invoice_pdf

11- Predicting Home Sale Prices Using Machine Learning (In Progress)

This project uses supervised machine learning to predict home sale prices with the Ames Housing dataset. It compares linear regression, ridge, lasso, and Random Forest models, focusing on prediction accuracy and how easy each model is to interpret.
